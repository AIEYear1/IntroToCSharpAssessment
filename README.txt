Introduction to CSharp Assesment
made in Visual Studios 2019.16.7.1 with Raylib-cs 3.1.5

To build, right-click the project in the solution explorer click publish then select folder profile and choose the empty folder you wish to build to and press publish.

A text-based Console RPG using Raylib for undertale esque attack scenes.

Controls are keyboard and mouse as specified in the game, type out the commands and type help to learn them.
